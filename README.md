# VidJot Project - Node.Js

A Node.js based app to list idea for YouTube video with authentication and full CRUD operation.

## Usage
If you are in local mode use:

```
yarn dev 
```
or

```
npm run dev 
```

## Instalation

Install all dependencies

```
yarn install 
```
or
```
npm install 
```

## Built With

* Node.Js
* Express.Js
* MongoDB
* Mongoose
* Bootstrap 4.1.3
* Handlebars
* Bcrypt
* JWT(JsonWebTokens)
